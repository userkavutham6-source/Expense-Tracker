A modern, data-driven web application that helps users track, categorize, and analyze their expenses with real-time insights and an intuitive dashboard.
✨ Features:
  ➕ Add, edit, and delete expenses
  📂 Category-based tracking (Food, Travel, Bills, etc.)
  📊 Real-time expense summaries
  📈 Interactive charts (category distribution, trends)
  🎯 Budget tracking with alerts
  🔍 Search and filter functionality
  💾 Persistent storage using localStorage
  📱 Fully responsive (mobile-first design)
  🌙 Dark & light mode support
 How It Works:
  Users input expense details (amount, category, date, description)
  Data is stored locally in the browser
  The app dynamically updates totals and visualizations
  Charts provide insights into spending patterns over time
🛠 Tech Stack:
  HTML5
  CSS3
  JavaScript (Vanilla / React if applicable)
⚙️ Usage:
  Add a new expense with details
  Categorize and track spending
  View charts and summaries
  Set budgets and monitor limits
  Filter or search past expenses

⚠️ Note: This is a frontend-only application and is not intended for storing sensitive financial data.
